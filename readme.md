# lorem-cli

> Lorem: Distributed Computing CLI

## Install

```
$ npm install --global lorem-cli
```

## Usage

```
$ lorem --help

  Usage
    $ lorem [input]

  Options
    --debug=[bool]  [Default: false]

  Examples
    $ lorem

    Config
    $ lorem config get
    $ lorem config set key.subkey val
    $ lorem config del key

```
